### Stitch Workflow (BUILD-WIRE-VERIFY Loop)

For each page (in priority order):

1. **STITCH** - Open `docs/STITCH-DESIGN-PROMPTS.md`
   - Copy the **Stitch Design Prompt** section
   - Paste into Google Stitch
   - Generate and iterate on design
   - Export `code.html` and `screen.png`
   - Save to `stitch-exports/[phase]/[page-name]/`

2. **CONVERT** - HTML to platform component:

   **Next.js:**
   - `class` → `className`, `<img>` → `<Image>`, `<a>` → `<Link>`
   - Add `'use client'` for interactive components

   **Vite:**
   - `class` → `className`, keep `<img>`, use React Router `<Link>`

   **Expo/React Native:**
   - `<div>` → `<View>`, `<p>` → `<Text>`, `<button>` → `<Pressable>`
   - `onClick` → `onPress`

   **Telegram:**
   - Use `@telegram-apps/telegram-ui` components
   - Add haptic feedback to handlers

   **Discord:**
   - Standard React + Discord SDK hooks

   See `BUILD-FROM-SCRATCH-WORKFLOW.md` for complete conversion rules per platform.

3. **WIRE** - Connect to backend:
   - Import hooks from `@/hooks/`
   - Import actions from `@/lib/actions/`
   - Import types from `@/types/`
   - Replace placeholder file with converted component

4. **AUDIT** - Verify all handlers connected:

   ```bash
   /audit PageName --fix
   ```

   - Check `docs/FUNCTIONAL-CONTRACT.md` for this page
   - All elements must show ✅ before proceeding
   - Fix any ❌ elements before moving on

5. **VERIFY** - Quick test:
   - `npx tsc --noEmit` passes
   - Page renders in browser
   - Data loads from hooks (check Network tab)
   - Forms submit and persist
   - All buttons trigger actions

6. **REVIEW** - Code review:

   ```bash
   git add src/app/[page]/page.tsx
   coderabbit review --staged --plain
   ```

7. **FIX** - Address any issues, then mark page complete

### Usage Example

```typescript
'use client';

import { useProjects } from '@/hooks/use-projects';
import { ProjectCard } from '@/components/project-card';

export function ProjectList() {
  const { data: projects, isLoading, error } = useProjects();

  if (isLoading) return <ProjectListSkeleton />;
  if (error) return <ErrorState error={error} />;
  if (!projects?.length) return <EmptyState />;

  return (
    <div className="grid gap-4 md:grid-cols-2 lg:grid-cols-3">
      {projects.map((project) => (
        <ProjectCard key={project.id} project={project} />
      ))}
    </div>
  );
}
```
