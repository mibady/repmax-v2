## **Design System: Color Palette**

Define the core color tokens for Tailwind and branding, ensuring consistency across components and layouts.

The Mystical Poet
This palette leans into the "elixir" concept—focusing on depth, magic, and an ethereal, twilight mood. It's sophisticated, modern, and feels like a secret discovered in a moonlit forest.

Core Idea: Deep, enchanting colors with a vibrant, living accent.

Best For: Indie, Alternative, Folk, or Experimental genres.

Name Token Hex Usage
Paper --color-paper #F9FAFB Primary backgrounds, giving a clean, soft feel.
Ink --color-ink #1F2937 Primary text, icons. A cool, deep charcoal that's softer than pure black.
Stone --color-stone #6B7280 Borders, secondary text, disabled states.
Moonlit Teal --color-moonlit-teal #14B8A6 Primary accent, CTAs, highlights. This is your "elixir"—vibrant, magical, and full of life.

## **Utility Classes for Consistency:**

- `.btn-primary`, `.btn-secondary`, `.btn-accent` - Consistent button styling
- `.card-glass`, `.card-solid`, `.card-gradient` - Card variants
- `.text-brand-primary`, `.bg-brand-primary` - Brand color applications
- `.gradient-text`, `.hero-glow` - Special effects using brand colors

✅ **Consistent Design**: All components use the same color tokens✅ **Easy Maintenance**: Change colors in one place, updates everywhere✅ **Brand Compliance**: Ensures all UI elements follow the design system✅ **Developer Experience**: Clear utility classes and helper functions
