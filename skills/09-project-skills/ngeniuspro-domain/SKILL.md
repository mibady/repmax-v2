# nGenius Pro Domain

## Product Overview

Digital agency platform:
- Agency website/portfolio
- Client project management
- Service offerings
- Lead capture and CRM
- Client portal

## Tech Stack

| Layer | Technology |
|-------|------------|
| Frontend | Next.js, TypeScript |
| Styling | Tailwind CSS, shadcn/ui |
| Backend | Supabase |
| Deployment | Vercel |

## Core Entities

- `clients` - Agency clients
- `projects` - Client projects
- `services` - Service offerings
- `leads` - Incoming inquiries
- `proposals` - Project proposals
- `invoices` - Billing
- `team_members` - Agency staff
- `portfolio_items` - Showcase work

## Key Features

1. **Portfolio** - Showcase completed projects
2. **Services** - List agency offerings
3. **Lead Capture** - Contact forms, CRM
4. **Client Portal** - Project status, files
5. **Proposals** - Generate and send proposals
6. **Invoicing** - Time tracking, billing
