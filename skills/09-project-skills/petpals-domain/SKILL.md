# PetPals Domain

## Product Overview

Pet social platform with:
- Social feeds for pet owners
- AI-powered pet advice and health tips
- Pet marketplace
- Community features and events
- Pet profiles and photo sharing

## Tech Stack

| Layer | Technology |
|-------|------------|
| Frontend | Next.js 15, React, TypeScript |
| Styling | Tailwind CSS, shadcn/ui |
| Backend | Supabase |
| AI | OpenAI GPT-4 |
| Sync | v0.app integration |

## Core Entities

- `users` - Pet owners
- `pets` - Pet profiles (name, breed, age, photos)
- `posts` - Social feed posts
- `comments` - Post comments
- `likes` - Post/comment likes
- `marketplace_listings` - Items for sale
- `events` - Community events
- `ai_conversations` - Pet advice chat history

## Key Features

1. **Pet Profiles** - Create profiles for pets with photos, breed info
2. **Social Feed** - Share updates, photos, milestones
3. **AI Pet Advisor** - Get pet care advice, health tips
4. **Marketplace** - Buy/sell pet supplies
5. **Community** - Events, groups, local meetups
