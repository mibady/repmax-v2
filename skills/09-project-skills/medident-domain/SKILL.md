# Medident Domain

## Product Overview

Medical/dental practice platform:
- Patient management
- Appointment scheduling
- Treatment planning
- Billing and insurance
- Patient communication

## Tech Stack

| Layer | Technology |
|-------|------------|
| Frontend | Next.js |
| Styling | Tailwind CSS, shadcn/ui |
| Backend | Supabase |
| Deployment | Vercel |

## Core Entities

- `practices` - Medical/dental practices
- `providers` - Doctors, dentists, hygienists
- `patients` - Patient records
- `appointments` - Scheduled visits
- `treatments` - Treatment plans
- `billing` - Invoices, payments
- `insurance_claims` - Insurance processing
- `communications` - Patient messaging

## Key Features

1. **Patient Portal** - Self-service booking, records access
2. **Scheduling** - Smart appointment management
3. **Treatment Plans** - Create and track treatments
4. **Billing** - Invoice generation, payment processing
5. **Insurance** - Claims submission, tracking
6. **Communications** - SMS/email reminders

## HIPAA Compliance

- Encrypted data at rest and in transit
- Audit logging for all PHI access
- Role-based access control
- BAA with Supabase
