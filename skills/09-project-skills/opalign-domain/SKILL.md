# Opalign Domain

## Product Overview

Operations alignment platform:
- Process documentation
- Team alignment tracking
- OKR/goal management
- Meeting coordination
- Decision logging

## Tech Stack

| Layer | Technology |
|-------|------------|
| Frontend | Next.js 15, React 19, TypeScript |
| Styling | Tailwind CSS, shadcn/ui |
| Backend | Supabase |
| Deployment | Vercel |

## Core Entities

- `organizations` - Companies/teams
- `teams` - Organizational units
- `processes` - Documented procedures
- `okrs` - Objectives and key results
- `meetings` - Meeting records
- `decisions` - Decision log
- `alignments` - Team alignment scores
- `action_items` - Follow-up tasks

## Key Features

1. **Process Docs** - Document and share procedures
2. **OKR Tracking** - Set and track objectives
3. **Meeting Hub** - Agendas, notes, action items
4. **Decision Log** - Track important decisions
5. **Alignment Score** - Measure team coordination
6. **Dashboards** - Operational health metrics

## Alignment Metrics

- **Process Adherence** - Following documented procedures
- **Goal Progress** - OKR completion rate
- **Meeting Effectiveness** - Action item completion
- **Decision Velocity** - Time to decisions
