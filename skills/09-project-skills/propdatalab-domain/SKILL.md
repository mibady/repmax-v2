# PropDataLab Domain

## Product Overview

Real estate data analysis platform:
- Property data aggregation
- Market trend analysis
- Investment scoring
- Comparable sales analysis
- AI-powered insights

## Tech Stack

| Layer | Technology |
|-------|------------|
| Frontend | Next.js 16 (App Router) |
| Backend | Supabase (PostgreSQL) |
| AI | OpenAI GPT-4 |
| Data | Property APIs, MLS feeds |
| Maps | Mapbox/Google Maps |

## Core Entities

- `properties` - Property records
- `markets` - Geographic markets
- `transactions` - Sales history
- `valuations` - Property valuations
- `analyses` - Investment analyses
- `reports` - Generated reports
- `watchlists` - Saved properties
- `alerts` - Market alerts

## Key Features

1. **Property Search** - Find properties with filters
2. **Valuation** - AI-powered property valuation
3. **Comps Analysis** - Comparable sales
4. **Market Trends** - Price trends, inventory
5. **Investment Score** - ROI predictions
6. **Reports** - Generate detailed reports

## Data Sources

- Public records (county assessor)
- MLS feeds
- Census data
- Economic indicators
- Permit data

## Investment Metrics

- Cap Rate
- Cash-on-Cash Return
- IRR (Internal Rate of Return)
- Price per Square Foot
- Days on Market
- Appreciation Rate
