# GovConX Domain

## Product Overview

Government contracting platform:
- Federal opportunity discovery
- Market intelligence
- Pipeline management
- Proposal tools
- Compliance tracking

## Tech Stack

| Layer | Technology |
|-------|------------|
| Frontend | Next.js, TypeScript |
| Styling | Tailwind CSS, shadcn/ui |
| Backend | Supabase |
| Data | SAM.gov API, USASpending |
| AI | GPT-4 for proposal assistance |

## Core Entities

- `companies` - Contractor profiles
- `opportunities` - Federal solicitations
- `pipelines` - Bid pipelines
- `proposals` - Proposal documents
- `contracts` - Awarded contracts
- `teaming` - Teaming partners
- `certifications` - Small business certs
- `past_performance` - Contract history

## Key Features

1. **Opportunity Search** - Find relevant RFPs/RFQs
2. **Smart Matching** - AI-matched opportunities
3. **Pipeline Tracker** - Manage bid/no-bid decisions
4. **Proposal Builder** - Templates, compliance matrix
5. **Teaming Network** - Find partners
6. **Analytics** - Win rate, market share

## Government Contracting Terms

- **SAM** - System for Award Management
- **NAICS** - Industry classification codes
- **PSC** - Product/Service codes
- **CAGE** - Contractor ID code
- **SDVOSB** - Service-Disabled Veteran-Owned
- **8(a)** - SBA disadvantaged program
- **HUBZone** - Historically underutilized zones

## Compliance Requirements

- SAM.gov registration
- DUNS/UEI number
- Small business certifications
- Past performance records
- Security clearances (if applicable)
