# Stitch Prompts - Discord

Templates for Discord Activities.

## Output Target

- Discord Activity components
- Discord color system
- Multi-user considerations

## Template

```markdown
## Screen: [ScreenName]

**Framework target:** Discord Activity

**Purpose:** [Screen purpose]

**Layout:**

- Responsive (300px - fullscreen)
- Works in sidebar and pop-out
- Dark mode primary

**Components:**

- Discord-styled buttons
- Cards with rounded corners
- [Specific components]

**Multi-user features:**

- [How multiple users interact]
- [Real-time updates needed]

**Copy:**

- Header: "[Text]"
- Description: "[Text]"
- Action: "[Button text]"

**States:**

- Connecting: Loading spinner
- Waiting: Waiting for others UI
- Active: Main interaction
- Complete: Results/summary

**Colors:**

- Primary: #5865F2 (Blurple)
- Background: #2f3136
- Text: #FFFFFF
```

## Conversion Notes

1. Use Discord color variables
2. Design for narrow sidebar first
3. Consider multiple participants
4. Handle connection states
5. Support real-time updates
