# Copywriting Skill

Neural-level persuasion copywriting for agency projects across all channels and verticals.

## Core Workflow

1. **Analyze the request** - Identify: channel, audience, awareness stage, price point, desired action
2. **Select frameworks** - Use auto-selection logic below OR honor explicit framework requests
3. **Consult references** - Read `references/frameworks.md` for framework details, `references/application-guide.md` for channel templates
4. **Generate copy** - Produce polished copy with framework reasoning and variations

## Output Format

Every deliverable includes:

```

[Primary copy version]

---

### Framework Reasoning
- **Primary Framework:** [Name] - [Why selected]
- **Psychological Triggers:** [List] - [How applied]

---

[Copy]

**Variation B: [Angle]**
[Copy]

[Copy]
```

## Framework Auto-Selection

### By Channel

| Channel | Default Stack |
|---------|---------------|
| Cold Email | PAS + Curiosity trigger + 1 Primal Desire |
| Landing Page | AIDA structure + Value Equation + Social Proof |
| Social Ads | PAS (condensed) + Scarcity OR Social Proof |
| Email Sequence | Progressive D.O.S.E. journey across emails |
| VSL/Video | AIDA + Star-Story-Solution + Emotional arc |
| Sales Page | 4 P's + Objection Hierarchy + Loss Aversion |
| Website Copy | FAB + Three Brains (features→benefits→justification) |

### By Awareness Stage

| Stage | Approach |
|-------|----------|
| 1 - Unaware | Educate, Gap Principle, no direct selling |
| 2 - Problem Aware | PAS, Loss Aversion, amplify pain |
| 3 - Solution Aware | BAB, differentiate, show transformation |
| 4 - Product Aware | 4 P's, handle objections, prove value |
| 5 - Most Aware | Scarcity, urgency, make buying easy |

### By Price Point

- **Low ticket (<$100):** Speed, simplicity, single CTA
- **Mid ticket ($100-$2K):** Value equation, social proof, urgency
- **High ticket ($2K+):** Long-form, story-driven, all 5 objections addressed

## Override Syntax

User can request specific frameworks:

- "Use PAS formula" → Apply PAS as primary structure
- "Apply loss aversion" → Lead with loss framing
- "Target primal desire #6" → Focus on Superiority & Winning
- "Stage 3 awareness" → Differentiation messaging

## Variation Generation

Always produce 3 variations testing different angles:

1. **Fear/Loss-based** - Loss aversion, consequences of inaction
2. **Desire/Gain-based** - Transformation, benefits, future state
3. **Status/Identity-based** - Social proof, belonging, superiority

## Ethical Guardrails

- Only claim substantiated results
- No artificial scarcity
- No exploitation of vulnerability
- Influence ethically; don't manipulate
- Long-term relationships over short-term conversions
